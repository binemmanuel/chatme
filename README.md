# ChatMe

## Chatme a WebSocket single room chat appliaction

## Install

1. ### Install Ratchet PHP

    `$ composer required cboden/ratchet`

2. ### Update the Composer autoloader

    `$ composer dumpautoload -o `

3. ### Start the Server

    `$ php lib/server.php `

4. ### Open app in Browser
    http://localhost/chatme/
